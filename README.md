# 🚦 Traffic Sign Recognition AI

## 📌 Project Overview
Traffic Sign Recognition AI is a deep learning-based application that detects and classifies traffic signs from uploaded images using a trained AI model.

## ✨ Features
- Detects traffic signs from uploaded images.
- Classifies multiple traffic sign categories.
- User-friendly Python interface.
- Deep Learning model for accurate prediction.
- Fast image processing.

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pillow
- Tkinter

## 📁 Project Structure
```
Traffic_Sign_Recognition_AI/
│── dataset/
│── models/
│── app.py
│── train_model.py
```

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/vinaygundeti/Traffic_Sign_Recognition_AI.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Train the model:
```bash
python train_model.py
```

4. Run the application:
```bash
python app.py
```
